LEVEL 1

1. Computer Essentials CST8101




This course teaches how to install, configure and troubleshoot problems with a basic understanding of the Windows Operating System. It also teaches computer organization systems such as basic numerical systems. And also, hardware and software components needed to run programs in a virtual machine.

2. Intro to Computer Programming CST8116




This course helps Students to develop an introduction to computer programming with emphasis on problem analysis and design. This course uses Java programming language; which is used as a means to implement problem-solving designs. Theory is reinforced with application by means of practical lab in the form of exercises , projects and assignment.

3. Intro to Database CST8215




The course includes fundamentals of relational databases and use Structured Query Language (SQL) to create, modify and query a database. It also familiarize students with the functions of a Database Management System (DBMS) and its components in comparison with legacy systems.

1. Web Programming CST8285




This course teaches student how to use HTML, CSS and Java script to build an attractive web-site.

2. Object Oriented Programming CST8284




Working as developer require indepth knoledge of Object Oriented Programming. In this course Students explore the basics of data structures and algorithms as well as basic Graphical User Interface (UI) programming. OOP concepts such as inheritance, abstraction and polymorphism are also covered.

3. Database Systems CST2355




This course covers advanced database concepts such as rollup and cube operations, metadata manipulation, data storage and retrieval. And it also teaches how to work with Microsoft Access and sql.
